Apa itu HTML?(hyper text language)

menurut wikipedia,HTML adalah bahasa markah standar untuk dokumen yang dirancang untuk di tampilkan di peramban internet. peramban internat menerima dokumen HTML dari server web atau dari penyimpanan lokal dan membuat dokumen menjadi halaman web multimedia. HTML menggambarkan struktur halaman web secara semantik dan isyarat awal disertakan untuk menampilkan dokumen

fungsi HTML:

  • fungsi utama HTML untuk membangun struktur halaman web,yang mencakup header,footer,navigasi,hingga kontennya.
  • agar tampilan web lebih menarik,melalui HTML dapat menyisipkan ilustrasi,vidio,grafik,dan media lainnya
  • HTML dapat berfungsi mengarahkan pengguna ke halaman atau website lain menggunakan fiturh yperlink.
  • HTML dapat membuat halaman web menyesuaikan ukuran dengan perangkat pengguna.

CSS

apa yang di maksud dengan CSS?

CSS adalah singkatan dari cascading tyle sheet yang dipergunakan untuk menentukan tampilan dan format pada sebuah website.Artinya,CSS dapat mengatur font,warna tulisan,dan latar belakang website.

fungsi dari CSS?

  • dapat mempercepat loading pada halaman website.
  • memudahkan dalam pengelolaan kode.
  • memberikan banyak variasi tampilan.
  • membuat website lebih enak dipandang
